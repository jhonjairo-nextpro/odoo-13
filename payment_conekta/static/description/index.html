SETUP

To start using conekta you need to add your Conekta Publick Key and Conekta Private Key to the configuration of the acquirer.
Go to: Website > Configuration > Payment Acquirers, then select Conekta and add your keys there.

To get more information about Conekta API Keys go to https://conekta.com
